## 📌 Project Overview

The **Attendance Portal** allows students to:

* Enter their details (Name, ERP, Year, Course)
* Mark attendance as ✅ Present or ❌ Absent
* View attendance records in a table
* Experience a clean UI with modal popup and blur effect

---

## ✨ Features

* 🎨 Modern black-themed UI
* 📱 Fully responsive design (mobile-friendly)
* 🧾 Student form with course selection
* 🪟 Popup modal for attendance marking
* 💾 Data stored using LocalStorage
* 📊 Dynamic attendance table
* 🖼️ Circular logo support
